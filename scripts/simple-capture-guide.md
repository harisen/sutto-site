# 🖼️ サイト全体画像化 簡単ガイド

## 最も簡単な方法（Chrome/Edge）

### 方法1: 開発者ツール（追加ソフト不要）

1. **キャプチャしたいサイトを開く**
   - 例: https://lp-service-site.vercel.app/samples/beauty-salon/

2. **F12キーを押して開発者ツールを開く**

3. **Ctrl + Shift + P を押す**（Macは ⌘ + Shift + P）

4. **「capture full」と入力**
   - 「Capture full size screenshot」が表示される

5. **Enterキーを押す**
   - 自動的にダウンロードフォルダに保存される（PNG形式）

6. **PNGをJPGに変換**（必要な場合）
   - Windowsの「ペイント」で開く
   - 「ファイル」→「名前を付けて保存」→「JPEG画像」

### 方法2: Chrome拡張機能

1. **GoFullPageをインストール**
   - Chromeウェブストアで「GoFullPage」を検索
   - 「Chromeに追加」をクリック

2. **キャプチャしたいサイトを開く**

3. **拡張機能アイコンをクリック**
   - 自動的にページ全体をスクロールしてキャプチャ

4. **保存形式を選択**
   - JPG/PNG/PDFから選べる

## 💡 Tips

### 画像サイズの調整
- 縦長すぎる場合は、画像編集ソフトで上部のメイン部分だけ切り取る
- SNS投稿用なら16:9や1:1にトリミング

### 複数画像の結合
- 複数のサンプルを1枚にまとめたい場合
- Canvaやペイントで横に並べる

### ファイルサイズの圧縮
- JPGで保存時に品質を80-90%に設定
- オンラインツール（tinypng.com など）で圧縮
