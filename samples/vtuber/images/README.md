# VTuber画像ファイル配置ガイド

このフォルダに以下の画像ファイルを配置してください。

## 配置済みファイル
- ✅ `logo.png` - 正方形ロゴ（フッター用）
- ✅ `logo-R.png` - 横長ロゴ（ヘッダー用）

## 必要な画像ファイル

### キャラクター画像（透過PNG推奨）
- `astra-main.png` - メインビジュアル（立ち絵）
- `astra-happy.png` - 喜び表情
- `astra-surprised.png` - 驚き表情
- `astra-wink.png` - ウインク表情
- `astra-cool.png` - クール表情
- `astra-singing.png` - 歌唱中表情
- `astra-sd.png` - SDキャラ版（オプション）

### 背景
- `bg-cyber.jpg` - サイバー都市背景
- `bg-pattern.png` - 星やデジタルパターン素材（オプション）

### 配信サムネイル
- `thumbnail-1.jpg` - 歌枠サムネイル
- `thumbnail-2.jpg` - ゲーム実況サムネイル
- `thumbnail-3.jpg` - 雑談枠サムネイル
- `thumbnail-4.jpg` - オリジナル曲サムネイル

## 推奨サイズ
- キャラクター画像: 400x600px以上
- 背景画像: 1920x1080px以上
- サムネイル: 1280x720px（16:9）

画像がない場合は、自動的にプレースホルダー画像が表示されます。
