=====================================
LP制作サービス - 納品ファイル設置マニュアル
=====================================

このドキュメントでは、納品されたLPファイルをWebサーバーに設置し、
インターネット上に公開する手順を説明します。

=====================================
1. 納品ファイルの構成
=====================================

納品フォルダには以下のファイル・フォルダが含まれています：

your-site/
├── index.html          # トップページ（メインのHTMLファイル）
├── css/               # スタイルシート用フォルダ
│   ├── reset.css      # ブラウザの標準スタイルをリセット
│   ├── style.css      # メインのスタイル定義
│   └── responsive.css # レスポンシブ対応用スタイル
├── js/                # JavaScript用フォルダ
│   ├── main.js        # メインのJavaScript
│   └── form.js        # フォーム機能用JavaScript（該当ページのみ）
├── images/            # 画像ファイル用フォルダ
│   └── （各種画像ファイル）
├── api/               # サーバーレス関数用フォルダ（メール送信機能使用時）
│   └── send-email.js  # メール送信処理
└── README.txt         # このファイル

※その他のHTMLファイル（service.html、pricing.htmlなど）も含まれる場合があります

=====================================
2. 公開に必要なもの
=====================================

LPを公開するには、以下の2つが必要です：

【必須】
・レンタルサーバー（月額500円〜2,000円程度）
  推奨：エックスサーバー、さくらインターネット、ロリポップなど
・独自ドメイン（年額1,000円〜5,000円程度）
  例：yourcompany.com、yourservice.jp など

【任意】
・SSL証明書（無料のLet's Encryptを推奨）
・FTPソフト（FileZilla、CyberduckなどFTPでアップロードする場合）

=====================================
3. 設置手順（レンタルサーバーの場合）
=====================================

3-1. レンタルサーバーの管理画面にログイン

3-2. ファイルマネージャーを開く
   ※サーバーによって「ファイル管理」「WebFTP」などの名称

3-3. public_html（またはwww、htdocsなど）フォルダを開く
   ※これがWebサイトのルートディレクトリです

3-4. 納品ファイルをすべてアップロード
   - フォルダ構造を保ったままアップロードしてください
   - index.htmlがルートディレクトリに配置されるようにします

3-5. パーミッション（権限）の確認
   - HTMLファイル：644（rw-r--r--）
   - フォルダ：755（rwxr-xr-x）
   ※通常は自動で適切に設定されます

3-6. ブラウザで確認
   http://あなたのドメイン/ にアクセスして表示を確認

=====================================
4. 設置手順（無料ホスティングサービスの場合）
=====================================

GitHub Pages、Netlify、Vercelなどを利用する場合：

4-1. サービスにサインアップ（無料）

4-2. 新しいプロジェクトを作成

4-3. 納品ファイルをアップロード
   - ドラッグ&ドロップまたはGit経由でアップロード

4-4. デプロイ（公開）を実行

4-5. 提供されたURLで確認
   ※独自ドメインの設定も可能（別途ドメイン取得が必要）

=====================================
5. SSL証明書の設定
=====================================

セキュリティとSEOのため、SSL証明書（https化）は必須です。

【レンタルサーバーの場合】
1. 管理画面から「SSL設定」を開く
2. 無料SSL（Let's Encrypt）を選択
3. 対象ドメインを選択して「設定」をクリック
4. 数分〜1時間程度で有効化されます

【設定後の確認】
https://あなたのドメイン/ でアクセスできることを確認

=====================================
6. メール送信機能の設定（オプション）
=====================================

お問い合わせフォームからメールを送信する場合：

【Formspreeを使用する場合】
1. https://formspree.io/ でアカウント作成
2. フォームを作成し、提供されたエンドポイントURLを取得
3. form.jsファイル内のURLを更新

【Resendを使用する場合（Vercelデプロイ時）】
1. https://resend.com/ でアカウント作成
2. APIキーを取得
3. Vercelの環境変数にRESEND_API_KEYを設定

=====================================
7. トラブルシューティング
=====================================

【ページが表示されない】
・index.htmlがルートディレクトリにあるか確認
・ファイル名の大文字小文字を確認（サーバーは大文字小文字を区別します）
・.htaccessファイルがある場合は内容を確認

【画像が表示されない】
・画像ファイルのパスを確認（images/フォルダ内にあるか）
・ファイル名の大文字小文字を確認
・画像ファイルの拡張子を確認（.jpg、.png、.webpなど）

【レイアウトが崩れる】
・CSSファイルが正しくアップロードされているか確認
・ブラウザのキャッシュをクリア（Ctrl+F5またはCmd+Shift+R）

【フォームが動作しない】
・JavaScriptファイルが正しくアップロードされているか確認
・ブラウザのコンソールでエラーを確認（F12キー）

10. 更新・メンテナンスについて
=====================================

【自分で更新する場合】
1. 該当するHTMLファイルをテキストエディタで開く
2. 内容を編集して保存
3. FTPまたはファイルマネージャーで再アップロード

【更新時の注意点】
・編集前にバックアップを取る
・文字コードはUTF-8を維持する
・HTMLタグを誤って削除しないよう注意
